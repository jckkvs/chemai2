# ⚡ ChemAI ML Studio クイックスタート

**5分で最初の分子物性予測を実行できます。**

---

## Step 1: セットアップ（初回のみ）

### Windows
```
setup.bat をダブルクリック
```

### Linux / macOS
```bash
chmod +x setup.sh && ./setup.sh
```

> 💡 全自動で仮想環境作成 → パッケージインストール → xTB設定まで行います。
> オプショナルパッケージが失敗してもアプリは動作します。

---

## Step 2: アプリ起動

### Windows
```
run_nicegui.bat をダブルクリック
```

### Linux / macOS
```bash
chmod +x run_nicegui.sh && ./run_nicegui.sh
```

ブラウザで **http://localhost:8085** を開きます。

---

## Step 3: サンプルデータで試す

1. 「📂 データ読込」タブで **「サンプルデータ」** をクリック
2. **Regression (SMILES)** を選択 → データが自動で読み込まれます
3. 「🧪 SMILES特徴量」タブに移動
4. **「計算開始」** ボタンをクリック → 記述子が自動計算されます
5. デフォルトのセット（🎯 汎用QSPR）が自動選択されます
6. 「🤖 解析実行」タブで **「解析開始」** をクリック
7. 結果タブでモデル性能・SHAP解釈を確認

---

## Step 4: 自分のデータで試す

1. CSV / Excel / SDF ファイルをドラッグ＆ドロップ
2. SMILES列と目的変数列を指定
3. 記述子エンジンを選択（プリセットからワンクリック）
4. 解析実行 → 結果確認

### 必要なデータ形式

| 列 | 内容 | 例 |
|----|------|-----|
| SMILES | 分子構造のSMILES表記 | `CCO`, `c1ccccc1` |
| 目的変数 | 予測したい数値 | 沸点, LogP, IC50 |
| （任意）その他 | 追加の説明変数 | 温度, 圧力 |
